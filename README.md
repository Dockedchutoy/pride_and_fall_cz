# Pride and Fall CZ // Pýcha a Pád (CZ)

Tohle je překlad komunitní módu pro HOI4 "Pride and Fall: The Last Days of Poland" (Česky "Pýcha a pád: Poslední dny Polska") do češtiny. 

Tohle je spíše osobní projekt, takže to bude trvat dobrou dobu, než to dokončím (pokud se tak vůbec stane).
