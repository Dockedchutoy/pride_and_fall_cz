# Pride and Fall CZ // Pýcha a Pád

This a translation of the HOI4 mod Pride and Fall: The Last Days of Poland into Czech. This is a personal project, so this might either take a while (a long time), or never get finished.

# I wanna help!

Feel free to put out a pull request. I'll check it out. Just, remember to check what has been already translated, so you don't waste 2 months on something somebody already did.

Only translations and corrections are accepted.
